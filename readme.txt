Waarom heb ik voor deze visualisatie en techniek?
Ik heb voor Chart js gekozen, omdat dit een van de populaire chart tools is. Ik vind het makkelijk in gebruik en erg overzichtelijk vergeleken andere chart tools. Ik heb vooral gebruik gemaakt van eenvoudige charts, omdat andere charts een beetje ingewikkeld worden, voor bijvoorbeeld familie en vrienden. Niet iedereen kan namelijk eenvoudig en snel iets vinden in charts als een sunburst etc.
